File Sharing

This project is a Python-based file-sharing application that allows users to transfer files quickly and securely over a network. Inspired by popular file-sharing tools like ShareIt, this system enables fast, peer-to-peer file transfer without requiring third-party dependencies or cloud storage. It is designed to work on local networks and can be expanded to support remote transfers with additional configurations.

Key Features:
✅ Peer-to-Peer File Sharing: Direct communication between sender and receiver without cloud dependency.
✅ Secure & Reliable Data Transfer: Ensures error-free transmission of files over a stable connection.
✅ Lightweight & Fast: No need for bulky installations; runs efficiently with minimal system resources.
✅ User-Friendly Interface: Simple command-line or GUI-based interaction for ease of use.
✅ Multi-Device Compatibility: Works across different operating systems (Windows, macOS, Linux).
✅ No Internet Required: Transfers files within the same network, making it ideal for offline sharing.

Technology Stack:
This project primarily utilizes Python’s socket programming to establish network communication between devices. The core technologies and libraries used include:
1️⃣ Programming Language
  🔹 Python – Chosen for its simplicity, cross-platform capabilities, and powerful networking libraries.
2️⃣ Networking & Communication
  🔹 Socket Library – Facilitates client-server communication for file transfer.
  🔹 TCP Protocol – Ensures reliable, ordered, and error-checked data transmission.
  🔹 IP Addressing & Port Handling – Configurable to establish stable network connections.
3️⃣ File Handling & System Interaction
  🔹 os & shutil Modules – Used for handling file operations like reading, writing, and storing files.
  🔹 Threading Module – Allows multiple connections simultaneously, enabling concurrent file transfers.
4️⃣ User Interface (Optional Enhancements)
  🔹 Tkinter or PyQt – (Planned) For a graphical user interface to enhance usability.
  🔹 Command-Line Interface (CLI) – Simple text-based interface for efficient operation.

How It Works:
1. The sender selects a file to share.
2. The receiver establishes a connection using the sender’s IP address and port.
3. A TCP socket connection is created between both devices.
4. The sender transfers the file in chunks to ensure smooth transmission.
5. The receiver receives, verifies, and saves the file to the specified directory.
6. Upon completion, both devices are notified of a successful transfer.
