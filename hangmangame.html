Hangman Game

The Hangman game is a Python-based graphical version of the classic Hangman game. It provides an interactive interface where players guess a randomly chosen word, one letter at a time. Correct guesses reveal the letters in the word, while incorrect guesses gradually complete the hangman figure. The game ends when the player either guesses the word correctly or loses after reaching the maximum number of incorrect attempts.

How It Works:
1. The game initializes by selecting a random word and displaying blank spaces for each letter.
2. The player enters a letter guess via the interface.
3. If the letter is in the word, it appears in the correct position(s); if not, a part of the hangman figure is drawn.
4. The game continues until the player either correctly guesses the word or loses by exceeding the allowed number of incorrect guesses.
5. A final message displays whether the player has won or lost, with an option to restart the game.
